==== Parameter/Optionen (Auswahl) ====

`-a`::
Architektur

`-d`::
Distribution bzw. Veröffentlichung

`-h`::
Host oder Server, von dem bezogen werden soll

`-s`::
Section

`-v`::
ausführliche Ausgabe (verbose)

`--method`::
Methode, die zum Bezug verwendet werden soll

`--nosource`::
keine Quellpakete, sondern nur Binärpakete

`--progress`::
Verlauf anzeigen

==== Beispielkonfiguration ====

* Kriterien:
** Architektur i386
** Distributionsbereiche 'main', 'contrib' und 'nonfree'
** Veröffentlichung 'Wheezy'
** nur Binärpakete
** Darstellung des Verlaufs

.Beispielaufruf für `debmirror` in `/home/ftp/debian`
----
# debmirror -a i386 -s main -s contrib -s non-free \
 -h ftp.debian.org \
 -d wheezy
 /home/ftp/debian \
 --nosource \
 --progress
# 
----
