=== debify ===

// Stichworte für den Index

* Ziel: 
** nimmt ein Verzeichnis voller Debian-Pakete und erstellt ein signiertes 
   und ordnungsgemäß strukturiertes Debian-Repository 
** Wrapper um `aptly` (siehe <<eigenes-apt-repository-anlegen-aptly>>)

* Download als Docker-Image <<debify-docker>>

* Quelle auf GitHub <<debify-github>>
