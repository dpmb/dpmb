=== Suche über Debtags ===

// Stichworte für den Index
(((Aptitude, Debtags-Browser)))
(((ähnliche Pakete finden, Suche mit Hilfe von Debtags)))
(((ähnliche Pakete finden, debtags)))
(((ähnliche Pakete finden, Debtags-Browser von Aptitude)))
Debtags (siehe <<erweiterte-paketklassifikation-mit-debtags>>) sind 
Paketmarkierungen analog zu Schlagworten, über die die Debianpakete 
klassifiziert sind. Je mehr übereinstimmende Schlagworte zwischen zwei 
Paketen bestehen, umso ähnlicher sind sie sich.

* Recherche über die Kommandozeile: `debtags search` 'Schlagwort'

* Nutzung über die TUI von `aptitude`: Debtags-Browser
